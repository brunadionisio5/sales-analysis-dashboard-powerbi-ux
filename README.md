# Relatório Financeiro com Foco na Experiência do Usuário - Power BI

## Sobre o projeto

Este projeto consiste na reformulação de um relatório financeiro desenvolvido no Power BI, com foco na melhoria da **experiência do usuário (UX)** e na organização visual das informações.

O objetivo foi transformar um relatório básico em um dashboard mais intuitivo, interativo e visualmente agradável.

---

## O que o projeto entrega

- Indicadores de desempenho de vendas
- Gráficos interativos
- Filtros para análise dinâmica
- Navegação entre páginas
- Detalhamento dos dados em matriz

---

## Melhorias aplicadas

- Reorganização dos elementos na tela
- Ajuste de cores e contraste
- Padronização do layout
- Inclusão de botões de navegação

---

## Como executar

1. Abra o arquivo `.pbix` no Power BI Desktop
2. Utilize os filtros para explorar os dados
